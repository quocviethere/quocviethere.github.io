Proof of valid PMF for symmetric Bernoulli distribution

The form of the Common Distributions > Bernoulli Distribution, given by:

p(x \vert \mu) = \left(\dfrac{1-\mu}{2}\right)^{(1-x)/2}\left(\dfrac{1+\mu}{2}\right)^{(1+x)/2}

f(x) =
\begin{cases}
1, \text{if } x= -1 \
0, \text{if } x= 1
\end{cases}

g(x) =
\begin{cases}
1, \text{if } x= 1 \
0, \text{if } x= -1
\end{cases}

\begin{aligned}
\sum{x=-1}^1 p(x \vert \mu) &= \sum{x=0}^1 \left(\dfrac{1-\mu}{2}\right)^{(1-x)/2}\left(\dfrac{1+\mu}{2}\right)^{(1+x)/2} \
&= \left(\dfrac{1-\mu}{2}\right)^{1}\left(\dfrac{1+\mu}{2}\right)^{0} + \left(\dfrac{1-\mu}{2}\right)^{0}\left(\dfrac{1+\mu}{2}\right)^{1} \
&= \left(\dfrac{1-\mu}{2}\right) + \left(\dfrac{1+\mu}{2}\right) \
&= 1
\end{aligned}
